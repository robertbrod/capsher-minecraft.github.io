Prism and Java
Install Prism Install the Prism Launcher Install Java 25 Install the JDK 25 (LTS) via Adoptium Ensure "Add to PATH" is checked during installation.

It is highly recommended to use the Prism Launcher and Java 25. Prism is highly configurable and does not require additional Java arguments. Java 25 is optimized and less memory intensive for GTNH.

Import the Instance
Install GT:NH Download the GT:NH Prism ZIP Import the Instance Drag the ZIP file directly into the Prism window

** Client version MUST match the server (v2.8.4). **

Setup and Join!
Setup Right-click > Edit > Settings > Java Set memory to 8GB and auto-detect Java 25 path Join the Server! Launch instance and enter the server IP
